    Predicts Survivability

    Overview

    1. Feature and Label Identification
  • Features: Select input variables (features) that may influence survivability, such as passenger class (Pclass), age, sex, number of siblings/spouses aboard (SibSp), number of parents/children aboard (Parch), ticket fare (Fare), and whether they embarked at a particular port (Embarked).
  • Label: The target variable is typically the Survived column, which indicates whether a passenger survived (1) or did not survive (0).

    2. Data Preprocessing
  • Data Cleaning: Handle missing data points (Age in this case) using techniques like imputation (e.g., mean for Age) or by dropping columns that does not affect the prediction such as ticket number, passenger name.
  • Encoding Categorical Variables: Transform categorical data (e.g., Sex, Embarked) into numerical representations using techniques such as one-hot encoding, making the data suitable for the machine learning algorithm.

    3. Model Development - Decision Tree Classifier Construction
  • Develop a model by using Decision Tree Classifier with hyperparameter criterion (gini and entropy), max_depth and evaluate model using cross validation.

    4. Model Optimization with Cross-Validation and Grid Search
  • Cross-Validation: Split the dataset into multiple subsets to train and validate the model on different partitions. This technique provides a robust estimate of model performance by testing the model across various data splits, typically using k-fold cross-validation.
  • Grid Search: Optimize hyperparameters by systematically evaluating different combinations (such as max_depth, min_samples_split, etc.) to find the best parameter set. The grid search is run over cross-validation folds to identify the configuration that yields the highest accuracy or other evaluation metrics (e.g., F1-score) for the model.

    5. Performance Evaluation and Refinement
  • Evaluation Metrics: Assess model accuracy and other relevant metrics (e.g. F1-scor) to ensure the classifier generalizes well to unseen data.
  • Feature Importance Analysis: Inspect which features the model deems important for survival prediction. Decision trees inherently allow for feature importance assessment, which helps identify which factors (like Sex and Pclass) are most predictive.

    6. Prediction
  • After refining, the model predicts passenger survivability on the test dataset with approximately 79% accuracy.
